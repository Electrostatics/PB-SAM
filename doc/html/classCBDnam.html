BDnam run function: generally what happens here:

  1. two molecules position and orientation are set
  2. one is in the center, the other at some random distance from it in the simulation box both have random orientations
  3. Then the timestep is set, and for the time that they are not STUCK, DOCKED or ESCAPED, a BD simulation advances
