The mainMakesphere function.

The function that uses a MC approach to coarse-graining the input. The function reads in the PQR and MSMS files. If this is a contact situation, the function then reads in the contact atoms and identifies the surface points they correspond to. It then identifies which spheres are solvent exposed and ensures that they are connected to a neighboring CG sphere. Then it prints out the CG spheres and terminates
